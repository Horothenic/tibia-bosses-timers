WHY?

I needed an easier-to-use loop timer for the bosses so I made a simple loop timer but specialized for each Tibia Bane boss.

Not web developer, I used Chat GPT and my basics in web to create this.

HOW TO USE?

1. Click the one you are currently farming and wait for the alarm.

2. When alarm sounds go kill the boss and return to your waiting zone.

GROUPS

If a boss is part of a group you can start individual timers for each boss of the group.
